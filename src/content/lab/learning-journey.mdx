## Tech Choices & Hosting Strategy

- **Portfolio site** → Keep using Next.js (modern, great for interactive demos)
- **Interactive visualizations** → Build in Next.js + TypeScript (Canvas/SVG)
- **CS50 exercises** → Maintain separate repos in C and Python, link from blog posts
- **Hosting flow**:
  1. Now → Vercel (fastest iteration)
  2. Later (AWS) → Amplify Hosting or S3 + CloudFront
  3. Later (Azure) → Static Web Apps or App Service
- **CI/CD** → Practice with GitHub Actions for all projects
