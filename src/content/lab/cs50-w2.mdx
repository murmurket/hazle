## `argc` and `argv[]`

`main()` can take arguments from the command line using:

```c
int main(int argc, string argv[])
```

- **argc (Argument Count)**: Number of command-line arguments (including the program name)  
- **argv (Argument Vector)**: Array of strings representing each argument

Example command:

```bash
./greet Alice Bob
```
